# Modern remake of sailplanner.nl using Svelte

```bash
npm install
npm run dev
```

Navigate to [localhost:5000](http://localhost:5000).

[Svelte API docs](https://svelte.dev/docs#script)
[Svelte tutorial](https://svelte.dev/tutorial/basics)
